# aiui
科大讯飞aiui后处理Demo，包括服务端、android app

# 准备工作
需要一台可以公网访问的服务器，并且安装nodejs，在AIUI控制台中配置后处理地址为该服务器地址

# 后处理服务端
```
npm install
node ./bin/www
```
